# Heart Place

당신이 좋아하는 모든 것들!

### Primarily Built with:

[next.js](https://github.com/vercel/next.js)  
[tailwind](https://github.com/tailwind)  
[turborepo](https://turbo.build/repo)  

---

### Install Dependencies

```bash
pnpm install
```

### Start the Development Server

```bash
pnpm run dev
```
